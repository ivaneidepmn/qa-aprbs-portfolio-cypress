# QA Portfolio – APRBS Certification Website

This project is a **QA portfolio project** created to demonstrate software testing practices using a real public website.

The project includes:

• Test planning  
• Test case design  
• Bug reporting  
• Test improvements suggestions  
• Automated tests using Cypress

Website used for testing:
https://qualidade.apprbs.com.br/certificacao
