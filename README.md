# 🐞 Problemas Identificados Durante os Testes

Durante a execução dos testes e análise exploratória foram identificados problemas que impactam a experiência do usuário.

---

## 1️⃣ Botão "Saiba mais" não funcional

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Alta  

### Descrição
O botão **"Saiba mais"** presente na primeira seção da página não executa nenhuma ação quando clicado.

### Impacto
Usuários não conseguem acessar informações adicionais sobre a certificação.

---

## 2️⃣ Botão "Avançar" do formulário não funciona

**Tipo:** Correção  
**Classificação:** Utilidade  
**Prioridade:** Alta  

### Descrição
Após preencher os campos do formulário de inscrição, o botão **"Avançar"** não executa a progressão para a próxima etapa.

### Impacto
Bloqueia completamente o fluxo de cadastro do usuário.

---

# 🔄 Integração Contínua (CI)

Este projeto possui pipeline automatizado utilizando **GitHub Actions**.

O pipeline executa automaticamente:

1️⃣ Instalação das dependências  
2️⃣ Execução dos testes Cypress  
3️⃣ Geração de evidências (vídeos e screenshots)  
4️⃣ Publicação dos artefatos

Isso garante validação contínua do sistema testado.
